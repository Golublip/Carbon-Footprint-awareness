# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-06-17

### Added
- **Carbon Footprint Tracker**: Support for logging Transportation, Electricity, Water, Food, and Shopping logs.
- **Carbon Calculator Engine**: Dynamic CO₂e math using configurable emission parameters.
- **AI Sustainability Coach**: Interactive chat interface with local fallback heuristics and optional Gemini API support.
- **Sustainability Challenges**: Interactive challenge cards, progress tracking, and complete check-in capabilities.
- **Analytics Dashboard**: Custom responsive SVG donut and line trend visualizations.
- **Accessibility Integration**: Focus ring indicators, aria landmarks, and chart-to-table toggles for screen readers (WCAG 2.1 compliance).
- **Achievements & Badges**: 9 milestone badges, streak trackers, and double canvas-confetti trigger events.
- **Weekly Carbon Reports**: Downloadable/printable weekly report cards.
- **6-Month Savings Forecast**: Multi-month future forecasting based on progressive habit adjustments.
- **Quality Assurance**: 32 unit and widget tests achieving >84% code coverage, with automated GitHub CI workflows.
